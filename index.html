# Start

Met deze plugin is het mogelijk om wapens toe te voegen aan jouw Minetopia server. Hiervoor heb je de Minetopia Texturepack (opens new window) nodig om de juiste textures te zien. Heb je extra hulp nodig? Voel je vrij om onze discord (opens new window) te joinen.

# Installatie

  • Download de plugin van de spigot pagina (opens new window).
  • Upload de plugin naar jouw server's plugins folder.
  • Restart jouw server.
  • De plugin zou nu moeten werken!
